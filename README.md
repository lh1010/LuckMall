LuckMall 是一款 100% 开放源码的 PHP 商城系统，基于 ThinkPHP5 开发，容易扩展，且具有强大的负载能力和稳定性。LuckMall 基于 Apache2.0 开源协议发布，免费且不限制商业使用。用户可以自由修改，打造符合自己意愿的电商平台。

LuckMall 系统设计思路清晰，代码整洁、可读性强，因此你可在 LuckMall 系统基础上很容易的扩展新功能。
